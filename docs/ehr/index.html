openEHR .NET

EHR Reference Model — Clarotech.OpenEHR.RM

A C# implementation of the openEHR RM Release 1.1.0 structural classes, covering the EHR, Composition, Common, Support, and Data Structures packages.

Depends on Clarotech.OpenEHR.RM.Datatypes — all DvXxx leaf values come from that package.

Installation

dotnet add package Clarotech.OpenEHR.RM
# Clarotech.OpenEHR.RM.Datatypes is pulled in automatically as a dependency
